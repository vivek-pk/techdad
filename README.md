TechDad — a minimal monochrome blog built with Next.js 15, Tailwind CSS, motion.dev, and Markdown posts.

## Getting Started

1. Install dependencies

```
npm install
```

2. Start the dev server

```
npm run dev
```

3. Open http://localhost:3000

## Content

- Write posts as Markdown files in `/posts` with frontmatter:

```
---
slug: my-post
title: My Post
date: 2025-08-13
excerpt: Optional short summary
---

Your content here.
```

## Deploy

This project is ready for Vercel. Push to GitHub and import in Vercel—no extra config needed.
